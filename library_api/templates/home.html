Api links

    Links to query forms

  • GET, POST 27.0.0.1:8000/api/v1/books/ - renders a form where the user can query the database for books
  • GET, POST 27.0.0.1:8000/api/v1/books/update_book/ - renders a form where the user can choose a book to update by id
  • GET, POST 27.0.0.1:8000/api/v1/books/delete_book/ - renders a form where the user can choose a book to delete by id
  • GET, POST 27.0.0.1:8000/api/v1/books/rate_book/ - renders a form where the user can choose a book to rate by id
  • GET, POST 27.0.0.1:8000/api/v1/books/reserve_book/ - renders a form where the user can choose a book to reserve by id
  • GET, POST 27.0.0.1:8000/api/v1/books/unreserve_book/ - renders a form where the user can choose a book to unreserve by id
  • GET, POST 27.0.0.1:8000/api/v1/books/add_book/ - shows a forms to add book to database on POST
  • GET, POST 27.0.0.1:8000/api/v1/books/update/[ID] - render a form to update an existing book in the database on ID

    Routes to endpoints

  • GET for the book model - returns a list of all the books in the system 27.0.0.1:8000/api/v1/books/
  • Post 27.0.0.1:8000/api/v1/books/[ID] - returns a book in the system by id
  • POST 27.0.0.1:8000/api/v1/books/delete_book/[ID] - deletes book in database by id
  • POST 27.0.0.1:8000/api/v1/books/[ID]/rate_book/[VALUE] - updates a retrived book's ratings depending on the values passed through
  • POST 27.0.0.1:8000/api/v1/books/[ID]/reserve_book/ - changes a books status to reserved if available
  • POST 27.0.0.1:8000/aapi/v1/books/[ID]/reserve_book/ - changes a books status to available if reserved
